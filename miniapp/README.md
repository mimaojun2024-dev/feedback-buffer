# 微信小程序版

这是从现有 `feedback-buffer` Web 版分出来的一条小程序实现线，放在仓库里的 `miniapp/` 目录下。

## 目录说明

- `miniapp/`：原生微信小程序代码
- `project.config.json`：微信开发者工具工程配置，`miniprogramRoot` 已指向 `miniapp/`

## 打开方式

1. 用微信开发者工具打开当前仓库根目录
2. 工程会自动识别 `miniapp/` 作为小程序根目录
3. 当前配置使用的是 `touristappid`，方便先预览和调结构
4. 真正发布前，把 `project.config.json` 里的 `appid` 换成你自己的

## 当前范围

已迁移为小程序 MVP 的内容：

- 首页状态入口
- 今日 / 月度 / 年度主轴
- 每日开始 / 收束 / 完成记录
- “我不想启动” 两分钟流程
- “我想看反馈” 23 秒缓冲流程
- “我需要勇气” 与记录列表
- “我在逃避”
- “我有点 down”
